Automated tests

We run your tests in parallel on reliable, clean devices using Appetize.io. Start your tests by uploading your APK with our script.

$ boop.py upload $APP_ID $PATH_TO_APK

That's it! Add this to your build script and we'll send you an email report for every build. You can replay your tests interactively to track down issues.
