A user-driven approach to test automation
for Android

Describe your tests visually

Your tests shouldn't need to know how your app is constructed. They describe the user experience, leaving developers free to build and refactor as they see fit.


tap('Username');
type_text('alice');

tap('Password');
type_text('ilovebob');

hide_keyboard();

tap(topmost({type: 'CheckBox'}));

tap('Sign in');

Continuous Integration made simple

Run tests in the cloud when you commit code, effortlessly. boop.js gives you a one-line integration into your existing CI system and notifies you of any failures immediately.

Test reports that matter

Test reports tell you how to reproduce the problem, give you the logs that matter and video of the test that failed: All the information you need for bug tracking.
